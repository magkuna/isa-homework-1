Kilka słów wprowadzenia

Nazywam się Magda. Dopiero zaczynam przygodę z programowaniem. Wszystkiego się jeszcze uczę i niektóre funkcje są dla mnie jeszcze nieznane. Każdy jednak kiedyś zaczynał i tworzył swoją pierwszą stronę. Mam nadzieję, że moje umiejętności będą się tylko pogłebiały, a kiedy zajrzysz do mnie znowu, nie poznasz wyglądu mojej strony. Pamiętaj! Nie ważne jak zaczynasz, ale ważne jak kończysz!!!

Technologie, których się obecnie uczę

HTML

HTML pozwala opisać strukturę informacji zawartych wewnątrz strony internetowej, nadając znaczenie poszczególnym fragmentom tekstu – formując hiperłącza, akapity, nagłówki, listy – oraz osadza w tekście dokumentu obiekty plikowe np. multimedia bądź elementy baz danych np. interaktywne formularze danych.

CSS

Język służący do opisu formy prezentacji (wyświetlania) stron WWW. CSS został opracowany przez organizację W3C w 1996 r. jako potomek języka DSSSL przeznaczony do używania w połączeniu z SGML-em. Pierwszy szkic CSS zaproponował w 1994 r. Håkon Wium Lie.

GIT

Rozproszony system kontroli wersji. Stworzył go Linus Torvalds jako narzędzie wspomagające rozwój jądra Linux. Git stanowi wolne oprogramowanie i został opublikowany na licencji GNU GPL w wersji 2.
